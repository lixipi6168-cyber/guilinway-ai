# Guilin Way MVP

Guilin Way is an English-first travel brand for overseas visitors who want an easier way to understand, plan, and explore Guilin.

Core positioning:

- Tagline: The easiest way to explore Guilin.
- Product positioning: Your AI Guide to Guilin.
- Core promise: Make Guilin Easy.
- Brand vision: Helping the world discover Guilin.

## Files

- `index.html` - brand homepage content and inquiry form
- `styles.css` - responsive page styling
- `script.js` - prepares a mailto travel question from the form
- `assets/guilin-hero.png` - hero image for the first viewport

## Preview

Open `index.html` directly in a browser, or serve the folder locally:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:4173/
```
